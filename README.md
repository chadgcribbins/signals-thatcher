# Signals - Thatchers Demo Project

This repository contains materials for demonstrating the Signals software using real hospitality stories from the Thatchers team.

## Contents

- **stories.md** - Collection of memorable hotel experiences shared by team members. These stories serve as source material for analyzing and extracting insights using Signals.

## Purpose

The stories in this repository capture various hospitality experiences - from luxurious stays to challenging situations. Signals will analyze these narratives to generate meaningful insights about guest experiences, service quality, and memorable moments.

## Getting Started

1. Clone this repository
2. Review the stories in `stories.md`
3. Use Signals to analyze and extract insights from these narratives
